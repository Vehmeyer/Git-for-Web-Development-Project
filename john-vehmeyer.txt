    1. What is Semantic HTML? 
    It's HTML that provides meaning to our markup
    2. What is HTML used for? 
    It's used to provide structure to our website
    3. What is an attribute and where do we put it?
    Attributes give additional information about HTML elements and can help configure elements. They are placed in opening tags. 
    4. What is the h1 tag used for? How many times should I use it on a page?
    The h1 tag is a header tag that is used for headings on websites. It should only be used once.
    5. Name two tags that have required attributes
    <a> tags and <img> tags
    6. What do we put in the head of our HTML document? 
    Metadata about the document - character set, title, etc.
    7. What is an id? 
    A selector that can be linked to one unique element. 
    8. What elements can I add an id to? 
    Any, but it typically shouldn't be used for styling since it can only be used once, where a class can be reused. 
    9. How many times can I use the same id on a page? 
    One time
    10. What is a class? 
    A group of elements, typically similar elements.
    11. What elements can I add a class to? 
    Any and as many as you want. 
    12. How many times can I use the same class on a page? 
    As many times as needed.
    13. How do I get my link to open in a new tab?
    In an <a> tag, include target="_blank"
    14. What is the alt attribute in the image tag used for? 
    To provide information about the image
    15. How do I reference an id?
    #idName
    16. What is the difference between a section and a div
    A div is a generic container, and is a block-level element, so it can take up a full line. A section is a generic sectioning element that can provide organization to your HTML.
